1. You compile the program with javac *.java

2. It is Search.java which contains the main() methode. To run the program type java Search

3. There are no spaces before or after the input (both dictionary and user). There is only one word per line in the dictionary.  

4. The BinarySearchTree class has more methodes than at first look looks necessary. This is because it is used by both the dictionary and by all the similar words.

5. Everything should work according to how it was described in the task.

6. The main core-structure of the tree is heavily influenced by the curricilum book:
Weiss, Mark Allen. "Data Structures and Algorithm Analysis in Java^TM". Essex, England: Pearsn Education Limited (2012) Third Edition.
